## 🚀 Quick Start

### Prerequisites
- Node.js 18+ 
- npm 9+

### Installation

```bash
# Clone the repository
git clone https://github.com/your-username/prime-connect-egypt.git
cd prime-connect-egypt

# Install dependencies
npm install

# Start development server
npm run dev
```

### Build for Production

```bash
npm run build
```

The output will be in the `dist/` directory as a single `index.html` file.

### Deploy to GitHub Pages

```bash
# Method 1: Using gh-pages
npm run deploy

# Method 2: Manual (recommended)
# Copy the `dist/` folder contents to your GitHub Pages branch
```

---

## 🌍 Deployment

### Option 1: GitHub Pages
1. Push code to `main` branch
2. Run `npm run build`
3. Go to repo Settings → Pages
4. Set source to `GitHub Actions` or `Deploy from a branch`

### Option 2: Any Static Host
Since the build outputs a single `index.html`, you can deploy to:
- Netlify (drag & drop `dist/`)
- Vercel
- AWS S3
- Any web server
